#无线电执照A证考试题库

作者: kare<br>
功能：在给定的题库361道题中顺序刷题或者随机刷题。<br>
版本：V1.0<br>
测试手机：<br>
	小米MIX2 <br>
	安卓版本8.0.0 <br>
	MIUI版本10.3<br>

子程序说明：<br>
	KRadioExamApp 手机APP源代码<br>
		编程语言：QT<br>
	KWXD_ExercisesSplit PC端代码，用于将当前程序目录下的exercises.txt解析并存储为json格式<br>
		编程语言：python<br>
